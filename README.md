# TechStore - Tienda Tecnologica
Proyecto que corresponde a pagina web para una tienda ficticia de tecnologia llamada techstore.

# Estudiante
Francisco Grandon

# Tecnologías utilizadas
- React
- Vite
- HTML5, CSS, JavaScript
- Node.js
- MongoDB Community Server
- Visual Studio Code
- Git / GitHub

# Caracteristicas de la web
- Header con logo, buscador, botones de sesión y redes sociales.
- Banner principal con imagen de fondo.
- Catálogo de productos conectado desde MongoDB.
- Carrito de compras con funciones como agregar, ver detalle, quitar productos y con cálculo de total.
- Sidebar con beneficios de la empresa.
- Footer con información de contacto y redes sociales.
- Diseño responsive para escritorio, tablet y celular.
- Operaciones CRUD sobre productos almacenados en MongoDB.

# Resumen
El proyecto permitió comprender en forma general la estructura de una pagina web que simula a una tienda online, como se organiza un trabajo tanto en frontend como backend ademas de como se conecta con una base de datos NoSQL como MongoDB. Permitió aplicar los conocimientos adquiridos y el uso de herramientas como React, Css, Git y Github comprendiendo lo fundamental de las buenas practicas en los flujos de trabajo de software.
